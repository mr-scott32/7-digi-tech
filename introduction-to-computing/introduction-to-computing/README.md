# Introduction to Computing

{% hint style="info" %}
## Learning Intention

* Identify appropriate hardware and software to develop design ideas and solutions
* Describe secure methods to share data and information safely online
{% endhint %}

{% hint style="success" %}
## Success Criteria

* I can identify hardware components of a PC
* I can explain the purpose of hardware
* I can explain the difference between primary and secondary storage
* I can save and share my files in a safe and secure manner
{% endhint %}
