# Hardware Activities

Remember to use the [**SEE Model**](../../../see-model.md) to answer these questions.



1. Outline function of a PC motherboard.

{% hint style="warning" %}
## Outline

Sketch in general terms. In other words, provide a brief overview of what function a motherboard performs.
{% endhint %}

2. Describe the purpose of a CPU.

{% hint style="warning" %}
## Describe

Provide characteristics and features. In other words, provide what a CPU is, what features it has, and what purpose it fulfills.&#x20;
{% endhint %}

3. Explain the difference between primary storage and secondary storage. Give an example of each.

{% hint style="warning" %}
## Explain

Relate cause and effect. Make the relationships between things evident. Provide why and/or how.

In other words, you not only need to explain what primary storage and secondary storage are, but also need to explain **how** they are different **and use an example of each to make this evident.**&#x20;
{% endhint %}
