# All About Data

{% hint style="info" %}
## Learning Intentions

* Describe secure methods to share data and information safely online
* Investigate data transmission and security through wired, wireless and mobile networks
* Explain ethical considerations for the ownership of data, information and artificial intelligence (AI) applications
{% endhint %}

{% hint style="success" %}
## Success Criteria

I can define data, information, networks and artifical intelligence

I can identify inputs, processes and outputs for basic data scenarios

I can explain how data is transmitted through wired, wireless and mobile networks

I can discuss ethics of data, information and AI

I can distinguish between Generative AI applications and online search engines

I can commit to responsible use of online search engines

I can commit to sourcing my information from trustworthy sources
{% endhint %}
