# Pseudocode Task

#### **Activity 1: Translate Everyday Task into Pseudocode**

Pick a simple real-life activity (e.g., making toast, brushing teeth, or packing a school bag). Write the steps in pseudocode using **BEGIN/END, GET, DISPLAY, IF, WHILE**.&#x20;

You must include **at least one if block** and **at least one while loop.**

**Here is an example - yours MUST be different (though can follow similar structure).**&#x20;

```
BEGIN
    GET bread
    IF bread is not mouldy THEN
        put bread in toaster
        WHILE bread not toasted
            wait
            GET toasted
        ENDWHILE
        spread butter
        DISPLAY "Toast is ready!"
    ELSE
        DISPLAY "Get new bread!"
    ENDIF
END
```

#### **Activity 4: Add a Loop**

Add a `WHILE` loop so the program keeps asking until the correct number is guessed.

```
BEGIN
    DISPLAY "Guess the number"
    GET guess
    IF guess = 7 THEN
        DISPLAY "Correct!"
    ELSE
        DISPLAY "Wrong!"
    ENDIF
END
```

#### **Activity 5: Write a Function**

Create a pseudocode function for a **common game action** (jumping, taking damage, rolling dice).

**Example of a function / subroutine:**

```
BEGIN roll_dice
    SET dice TO RANDOM 1–6
    DISPLAY "You rolled", dice
END roll_dice
```
