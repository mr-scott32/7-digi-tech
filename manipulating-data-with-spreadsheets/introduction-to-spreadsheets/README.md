# Introduction to Spreadsheets

{% hint style="info" %}
## Learning Intention

* Select data, information, tools, systems and technologies to make digital solutions and projects
* Design user interfaces (UI) considering the user experience (UX) of digital systems
* Evaluate authenticity, accuracy and timeliness of data and information
{% endhint %}

{% hint style="success" %}
## Success Criteria

I can reserach, select, process and visualise data in an Excel spreadsheet

I can design spreadsheets which are appropriately formatted to consider user experience (UX)

I can evaluate data and information in terms of authenticity, accuracy and timeliness
{% endhint %}
